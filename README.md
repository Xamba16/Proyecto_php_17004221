# Proyecto PHP
## Descripción 
Este repositorio contiene una aplicación web básica desarrollada en PHP. El proyecto despliega un encabezado con el nombre del autor y una imagen alojada localmente en el servidor. Es un ejercicio fundamental para entender la comunicación entre el sistema de archivos del servidor y el navegador del cliente.
## Prerrequisitos (tecnologías a instalar)
Para visualizar este proyecto, el usuario debe contar con:
  Servidor Web: Un entorno con soporte para PHP como XAMPP, WAMP o un servidor Linux con Apache/Nginx instalado.
  PHP: Versión 7.4 o superior instalada y configurada.
  Navegador Web: Chrome, Firefox o cualquier navegador moderno para acceder a la dirección del servidor.
## Pasos para su ejecución
1.  Clonar el repositorio: Descarga los archivos en la carpeta raíz de tu servidor.
2.  Organización de archivos: Asegúrate de que el archivo y la imagen se encuentren en el mismo directorio.
3.  Iniciar el servidor: Verifica que el servicio de Apache o el servidor PHP esté activo.
4.  Acceso: Abre el navegador e ingresa la ruta correspondiente. Si es un servidor local, usa http://localhost/proyecto_php/index.php o la dirección IP asignada, como http://192.168.0.101/proyecto_php/index.php.
